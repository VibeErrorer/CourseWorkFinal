СТРУКТУРА ПРОЕКТА
=================

src/
  main.cpp              — сотрудники, задачи дня, запуск Simulate()
  core/
    company.*           — контейнер, цикл рабочего дня (8 сек реального времени)
    simulation_time.h   — 1 симулированный час = 1 реальная секунда
  employees/
    boss.*              — назначение задач, лог, сбор невыполненных
    employee.*          — занятость, текущая задача
    auto_memory_doll.*  — печать писем
    mail_man.*          — доставка
  events/
    event.*             — PrintRequest, DeliveryRequest, BusinessTrip
  statistics/
    statistic.*         — счётчики за день

СИМУЛЯЦИЯ
=========
- Рабочий день: 8 симулированных часов = 8 реальных секунд
- Босс берёт задачи из очереди, ждёт свободного сотрудника
- По завершении — статистика; в конце дня — невыполненные в hodgins_journal.txt

ИСТОРИЯ
=======
Почтовая компания Клавдия Ходжинса из сериала Виолета Эвергарден.
Компания занимается придоставлением услуг авто запоминающих кукол
(секретарей с выездом на место или для работы в офисе), а также доставкой писем.
